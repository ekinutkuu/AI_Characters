# AI Characters

**Currently under development**


## Dependencies:

 - Node v20.17.0
 - React v18.3.1
 - npm v10.8.3

## Getting Started

```
git clone https://github.com/ekinutkuu/AI_Character
```

```
cd AI_Characters
cd frontend/
npm install
npm start
```

```
cd AI_Characters
cd backend/
npm install
node server.js
```

Project will start at:
[http://localhost:3000](http://localhost:3000/)
